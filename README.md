# ClarityAI — Free AI Accounting for Everyone

A free AI-powered accounting assistant and expense tracker built with React and Claude AI.

## Features
- AI chat assistant for accounting, tax, and budgeting questions
- Expense tracker with category breakdown and visual stats
- AI-powered spending analysis and financial insights
- 6 languages: English, Spanish, French, Arabic, Hindi, Chinese
- Fully accessible for everyone
- Your API key never leaves your browser

## How to Use
1. Get a free API key from console.anthropic.com
2. Paste it into the API key bar at the top
3. Start chatting or tracking expenses!
